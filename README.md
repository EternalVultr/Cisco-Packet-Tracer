# Cisco Packet Tracer: Integrated Enterprise Network

Proyek ini adalah tugas akhir dari pelatihan jaringan intensif 7 hari. Topologi ini mengintegrasikan berbagai teknologi networking dalam satu infrastruktur utuh.

### Fitur Utama:
- **Layer 2:** VLAN, VTP Server-Client, LACP EtherChannel, dan Port Security.
- **Layer 3:** Inter-VLAN Routing (Router-on-a-Stick) dan DHCP Server.
- **WAN & Security:** NAT Overload (PAT) untuk akses internet dan Standard/Extended ACL.

### Verifikasi Keberhasilan:
Keberhasilan NAT dibuktikan dengan tabel translasi saat perangkat di jaringan lokal melakukan ping ke IP Publik (200.1.1.2).
